格式化字符串

各符号表示：

%d - 整数
%s - 字符串
%f  - 浮点数
%x - 十六进制整数

实例

print('%2d-%02d' % (3, 1))
print('%.2f' % 3.1415926)

解释：

%后数字若为一位表示位数，如若两位，第一位是补充数字，第二位是位数
浮点数前加.
%%表示转义为%
