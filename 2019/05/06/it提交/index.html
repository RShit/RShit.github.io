git提交步骤

  • git init //初始化仓库
  • git add .(文件name) //添加文件到本地仓库
  • git commit -m “first commit” //添加文件描述信息
  • git remote add origin + 远程仓库地址 //链接远程仓库，创建主分支
  • git pull origin master // 把本地仓库的变化连接到远程仓库主分支
  • git push -u origin master //把本地仓库的文件推送到远程仓库

强制提交

如果确定代码无误，可以强制提交
  • git push -f
